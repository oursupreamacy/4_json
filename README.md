# Prettify JSON

Prettifies json-files

# Quickstart

Just launch it from terminal.

Example of script launch on Linux, Python 3.5:

```bash

$ python pprint_json.py <path to file>
[
    {
        "geoData": {
            "coordinates": [
                37.39703804817934, 
                55.740999719549094
            ]
    }
]

```
